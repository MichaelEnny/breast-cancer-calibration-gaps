## Core Argument

Healthcare AI governance frameworks were designed for single-inference static models. Agentic AI — autonomous systems making sequential decisions in clinical settings — requires governance at the pipeline level. This paper proposes a five-layer trustworthiness governance framework and argues that blockchain/DLT provides the accountability infrastructure to make it verifiable.

**The five layers:**
1. Reliability Evaluation (calibration, conformal prediction uncertainty)
2. Fairness Auditing (subgroup calibration equity)
3. Provenance Verification (model lineage, on-chain registry)
4. Consent Management (smart contracts for patient-controlled AI action scope)
5. Deployment Readiness (pre-deployment checklist + ongoing monitoring)
